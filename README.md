
# RN-Awesome-Loader
Make awesome grid like activity indicators

## Installation

Use the package manager [npm](https://npmjs.com) to install RN-Awesome-Loader.

```bash
npm install rn-awesome-loader
```

## Usage

```javascript
import AwesomeLoader from 'awesome-loader'

...
render () {
 return (
  <View>
    ....
    <AwesomeLoader />
  </View>

 )
}
...
 
```
